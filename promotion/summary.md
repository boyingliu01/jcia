# JCIA 推广材料总览

---

## 推广材料清单

本目录包含JCIA（Java Code Impact Analyzer）项目的完整内部推广材料，帮助公司同事了解和使用这款工具。

### 文件结构

```
promotion/
├── poster_content.md          # 一页海报设计内容
├── email_template.md           # 推广邮件模板
├── demo_script.md             # 演示脚本
├── faq.md                     # 常见问题解答
├── quickstart.md              # 快速开始指南
└── summary.md                 # 推广材料总览（本文件）
```

## 各文件说明

### 1. poster_content.md - 一页海报设计内容

**用途：** 提供海报设计的完整文字内容，可直接交给设计团队制作海报。

**包含内容：**
- 海报标题和副标题
- 核心价值（一句话描述、3大价值）
- 5大关键特性
- 工作流程示意图（文字版）
- 4个使用场景
- 效果对比表格
- 开始使用指引
- 视觉设计建议（颜色、字体、图标）

**使用场景：**
- 制作宣传海报张贴在公司公共区域
- 制作PPT用于技术分享会
- 制作易拉宝用于活动展示

---

### 2. email_template.md - 推广邮件模板

**用途：** 提供3套完整的推广邮件模板，可直接发送或稍作修改后使用。

**包含模板：**

**模板一：首次发布通知**
- 主题：[新工具发布] JCIA - Java代码影响分析专家
- 适用场景：产品首次正式发布
- 包含内容：产品介绍、核心价值、关键特性、使用场景、快速开始、联系方式

**模板二：功能更新通知**
- 主题：[功能更新] JCIA v2.0 发布 - 新增混合分析融合引擎
- 适用场景：新版本功能更新
- 包含内容：更新亮点、升级指南、新配置说明

**模板三：邀请试用通知**
- 主题：[诚邀试用] JCIA内测招募 - 体验智能化代码影响分析
- 适用场景：招募内测用户
- 包含内容：产品介绍、参与条件、报名方式、激励机制

**邮件发送建议：**
- 发送时机和频率
- 发送对象选择
- 发送渠道建议

## 使用建议

### 推广策略建议

1. **分阶段推广**
   - 第一阶段：种子用户招募（使用email_template.md中的模板三）
   - 第二阶段：小范围内测（使用demo_script.md进行演示）
   - 第三阶段：全面推广（使用email_template.md中的模板一和poster_content.md制作宣传材料）

2. **多渠道推广**
   - 邮件通知（使用email_template.md）
   - 海报张贴（使用poster_content.md）
   - 技术分享会（使用demo_script.md）
   - 内部Wiki文档（使用quickstart.md和faq.md）

3. **持续支持**
   - 建立技术支持渠道
   - 定期收集用户反馈
   - 持续更新FAQ文档
   - 提供培训和分享

### 针对不同受众的推广重点

| 受众 | 推广重点 | 推荐材料 |
|------|---------|----------|
| 开发工程师 | 日常开发效率提升、提交前自检 | quickstart.md, faq.md |
| 测试工程师 | 测试用例选择、回归测试优化 | demo_script.md (测试选择部分) |
| 技术负责人 | 团队效率提升、CI/CD集成 | poster_content.md, email_template.md |
| 架构师 | 影响分析能力、系统风险评估 | faq.md (架构相关部分) |
