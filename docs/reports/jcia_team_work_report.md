## 一、项目概述

JCIA (Java Code Impact Analyzer) 是一个 Python 工具，用于分析 Java 代码变更的影响范围，智能选择需要运行的测试用例，并提供回归分析能力。

### 原始需求
1. 变更代码影响分析 - 自动分析 Git 提交，识别变更的文件和方法
2. 影响图构建 - 构建调用链图，计算受影响的类和方法
3. 智能测试选择 - 基于 STARTS 算法和影响范围选择测试用例
4. AI 测试生成 - 集成 LLM 服务，自动生成测试用例建议
5. 多格式报告 - 支持 HTML、JSON、Markdown 格式报告
6. 回归分析 - 对比基线测试和回归测试结果，识别回归问题
